## 🎯 项目目标与价值 / Project Goals & Value

### 为什么创建这个知识库？/ Why Create This Knowledge Base?
- **信息整合 / Information Integration**：将分散的课程信息、学习经验集中整理，节省同学们的搜索时间 / Centralize scattered course information and learning experiences to save students' search time
- **经验传承 / Experience Inheritance**：让学长学姐的学习心得能够有效传递给后续学弟学妹 / Enable effective transfer of learning insights from seniors to junior students
- **学习指导 / Study Guidance**：提供系统性的课程规划和复习重点，帮助提高学习效率 / Provide systematic course planning and review focus to improve learning efficiency
- **资源共享 / Resource Sharing**：汇集优质的学习资源和参考材料 / Aggregate high-quality learning resources and reference materials

## 📖 使用指南 / Usage Guidelines

### ✅ 推荐用法 / Recommended Usage
- **课程预习 / Course Preview**：提前了解课程内容和难度，制定学习计划 / Understand course content and difficulty in advance, make study plans
- **复习参考 / Review Reference**：查看重点知识汇总，优化复习策略 / Review key knowledge summaries, optimize review strategies
- **经验借鉴 / Experience Learning**：学习前辈的学习方法和注意事项 / Learn from predecessors' study methods and precautions
- **资源查找 / Resource Finding**：获取推荐教材、工具和在线资源链接 / Obtain recommended textbooks, tools and online resource links

### ❌ 严禁行为 / Prohibited Behaviors
- **直接抄袭 / Direct Plagiarism**：将仓库中的代码、报告直接提交为个人作业 / Directly submit repository code or reports as personal assignments
- **考试作弊 / Exam Cheating**：在任何形式的考试中使用本仓库内容 / Use repository content in any form of examination
- **代写服务 / Ghost Writing**：利用本仓库为他人代写作业或项目 / Use repository to write assignments or projects for others
- **商业转载 / Commercial Reproduction**：未经授权将内容用于培训机构、付费课程等 / Use content for training institutions or paid courses without authorization

## 🤝 如何贡献 / How to Contribute

### 贡献方式 / Contribution Methods
- **📝 内容贡献 / Content Contribution**：补充课程笔记、整理重点知识 / Supplement course notes, organize key knowledge
- **🔗 资源推荐 / Resource Recommendation**：分享优质的学习资源和工具 / Share high-quality learning resources and tools

- **原创性 / Originality**：确保贡献的内容为原创或已获得授权 / Ensure contributed content is original or authorized
- **准确性 / Accuracy**：信息准确，对其他同学负责 / Information accurate, responsible to other students
- **完整性 / Completeness**：提供充分的context和说明 / Provide sufficient context and explanation
- **时效性 / Timeliness**：及时更新过时的信息 / Update outdated information promptly

## ⚠️ 免责声明与学术诚信 / Disclaimer & Academic Integrity

### 📚 项目定位 / Project Positioning
本知识库旨在为 BDIC 软件工程专业学生提供**学习辅助与经验分享**，帮助大家更好地理解课程内容、规划学习路径。  
This knowledge base aims to provide **study assistance and experience sharing** for BDIC Software Engineering students, helping everyone better understand course content and plan learning paths.

> **重要提醒：本仓库是学习指导工具，不是作业答案库！**  
> **Important Reminder: This repository is a study guidance tool, not an assignment answer bank!**

### 🚫 零容忍政策 / Zero Tolerance Policy
我们对以下学术不端行为采取**零容忍**态度：  
We adopt a **zero tolerance** attitude towards the following academic misconduct:

- **直接抄袭 / Direct Plagiarism**：复制本仓库内容用于课程作业、实验报告或考试提交 / Copy repository content for course assignments, lab reports or exam submissions
- **变相抄袭 / Disguised Plagiarism**：仅修改变量名、格式后提交为个人作业 / Only modify variable names or formatting before submitting as personal work
- **商业滥用 / Commercial Abuse**：未授权将本仓库内容用于商业或盈利用途 / Use repository content for commercial or profit purposes without authorization
- **学术造假 / Academic Fraud**：使用本知识库参与任何形式的学术不端行为 / Use this knowledge base to participate in any form of academic misconduct

### ⚖️ 后果警告 / Consequence Warning
违反学术诚信的后果包括但不限于：  
Consequences of violating academic integrity include but are not limited to:
- **课程处分 / Course Penalties**：成绩取消（F/NG）、学术警告、严重者退学 / Grade cancellation (F/NG), academic warning, expulsion for serious cases
- **个人信誉 / Personal Reputation**：学术声誉受损，影响升学和就业 / Academic reputation damage, affecting further education and employment
- **法律风险 / Legal Risks**：知识产权侵权，可能面临法律诉讼 / Intellectual property infringement, potential legal action

### 🛡️ 监督措施 / Supervision Measures
- 使用技术手段检测代码相似度 / Use technical means to detect code similarity
- 配合学校学术诚信调查 / Cooperate with school academic integrity investigations
- 保留向学院教学/学术委员会报告的权利 / Reserve the right to report to college teaching/academic committees

> 📌 **请理性使用本资源，尊重知识劳动，共同维护良好的学术环境。学术诚信，从我做起！**  
> 📌 **Please use this resource rationally, respect intellectual labor, and jointly maintain a good academic environment. Academic integrity starts with me!**
