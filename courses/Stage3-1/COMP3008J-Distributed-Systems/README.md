## 课程介绍

分布式系统是现代计算机科学的核心领域，研究如何设计和实现运行在多台计算机上的系统。涵盖系统架构、通信机制、一致性、容错等关键技术。

一共有3个Tut，内容是Final考试的例题，可以针对Tut来做整体的复习。

Project是一个案例研究，调研写报告。

### 核心知识点
- 分布式系统模型与架构
- 通信机制 (RPC, 消息传递)
- 时间同步与分布式算法
- 一致性与复制 (CAP定理)
- 容错机制与分布式事务
- 分布式存储与云计算
